# LightRoll_Cleaner

写真・動画ライブラリのクリーンアップツール

## 概要

LightRoll_Cleanerは、iOSデバイスの写真・動画ライブラリを整理・最適化するためのアプリケーションです。

## 機能

- 重複ファイルの検出と削除
- 類似画像のグルーピング
- 大容量ファイルの特定
- ストレージ使用状況の可視化

## 技術スタック

- Swift
- SwiftUI
- Photos Framework

## プロジェクト構造

```
LightRoll_Cleaner/
├── docs/           # ドキュメント
│   ├── CRITICAL/   # 重要ドキュメント
│   ├── modules/    # モジュール仕様
│   └── archive/    # アーカイブ
├── src/            # ソースコード
│   └── modules/    # モジュール
└── tests/          # テストコード
```

## セットアップ

```bash
# リポジトリをクローン
git clone <repository-url>
cd LightRoll_Cleaner

# Xcodeでプロジェクトを開く
open LightRoll_Cleaner.xcodeproj
```
