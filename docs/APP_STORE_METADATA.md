## 3. 説明文（4000文字以内）

### 日本語版

```
LightRoll Cleanerは、iPhoneの写真ライブラリを自動分析し、重複写真・類似写真・ブレた写真を検出する写真整理アプリです。不要な写真を見つけて削除し、ストレージ容量を節約できます。

すべての写真分析はデバイス内で完結するため、あなたの写真が外部サーバーに送信されることはありません。プライバシーを守りながら、効率的に写真を整理できます。

【主な機能】

■ 類似写真の自動検出
AI画像分析技術により、見た目がそっくりな写真を自動でグループ化。各グループからベストショットを自動選定し、残りの不要な写真を簡単に削除できます。

■ 重複写真の検出
完全に同じ写真や、ほぼ同一の写真を検出します。知らないうちに増えた重複写真をまとめて整理できます。

■ ブレ写真の検出
手ブレやピンボケで撮影に失敗した写真を自動検出。不要なブレ写真を素早く見つけて削除できます。

■ セルフィー・スクリーンショットの分類
セルフィーやスクリーンショットを自動識別し、カテゴリ別に整理。必要に応じてフィルタリングが可能です。

■ ゴミ箱機能（30日間復元可能）
削除した写真は30日間ゴミ箱に保管されます。誤って削除した写真もワンタップで復元できるので安心です。

■ ストレージ使用状況の可視化
写真ライブラリのストレージ使用量をわかりやすく表示。どれだけ容量を節約できるか一目でわかります。

■ カスタマイズ可能な設定
- 類似度の閾値調整
- 表示列数の変更（2〜6列）
- 並び順の変更（日付順・容量順）
- 通知設定（スキャン完了通知・リマインダーなど）

■ 日本語・英語対応
デバイスの言語設定に応じて自動で言語を切り替え。設定画面から手動で変更することも可能です。

【料金プラン】

無料でも基本的な写真整理機能をお使いいただけます。

● 無料プラン
- 初回スキャン可能
- 累計50枚まで削除可能
- 広告表示あり

● プレミアムプラン
- 無制限スキャン
- 無制限削除
- 広告非表示
- 月額・年額・買い切りから選択可能
- 月額プランは7日間の無料トライアル付き

【サブスクリプションについて】
- 月額プラン: 確認後、iTunesアカウントに課金されます
- 年額プラン: 確認後、iTunesアカウントに課金されます
- 現在のサブスクリプション期間が終了する24時間前までにキャンセルしない限り、自動更新されます
- サブスクリプションの管理・キャンセルは、購入後にアカウント設定から行えます
- 無料トライアル期間中にサブスクリプションを購入した場合、未使用のトライアル期間は無効となります

プライバシーポリシー: アプリ内「設定」>「プライバシーポリシー」
利用規約: アプリ内「設定」>「利用規約」
```
