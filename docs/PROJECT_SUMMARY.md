# LightRoll Cleaner - Project Summary

**LightRoll Cleaner**は、iOSデバイスのカメラロールから不要な写真を自動検出し、効率的に削除できる写真クリーナーアプリです。Vision/CoreMLを活用した画像分析により、類似写真・自撮り・スクリーンショット・ブレ写真を自動グルーピングし、ベストショットを提案。ユーザーは不要な写真をまとめて選択・削除でき、端末のストレージ容量を一気に解放できます。

**主要機能**: 写真スキャン＆自動グルーピング / ベストショット提案 / 一括削除 / 容量ダッシュボード / ゴミ箱（30日復元可能） / 空き容量アラート / 定期リマインド

**技術スタック**: Swift 5.9+ / SwiftUI / Photos Framework / Vision / StoreKit 2 / Google Mobile Ads

**アーキテクチャ**: MVVM + Clean Architecture / 9モジュール構成（118タスク / 約190時間）

**ビジネスモデル**: Free（1日50枚削除上限・広告あり） / Premium Monthly ¥480 / Premium Yearly ¥3,800 / Lifetime ¥9,800
